Run the Generate → Destroy loop on a specific element to discover
taste principles through systematic rejection.

1. Spawn @agent-taste-destroyer on the target element.
2. Generate 8-10 genuinely distinct versions.
3. Destroy each with structural critiques.
4. Synthesize the discovered taste principle.
5. Rebuild one surviving version.
6. Log principle to .claude/memory/taste/principles.md
7. Log all rejections to .claude/memory/taste/rejection-log.md

This is a TRAINING exercise. The point is the rejection vocabulary
and discovered principles, not just the final version.

Target element: $ARGUMENTS
